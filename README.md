## PowerPlant
<img width="1920" height="1080" alt="obraz" src="https://github.com/user-attachments/assets/f41ee68c-759f-453b-8cda-163a0fb26310" />

### Przeznaczenie \ Purpose
**EN:** The application simulates the operation of 20 power generators that send data at short intervals on the power generated, the data is stored in a MongoDB database and once a day a report is generated from all the generators with the average power generated by each generator during the hour.

**PL:** Aplikacja symuluje działanie 20 generatorów energii, które w krótkich odstępach czasu wysyłają dane dotyczące wygenerowanej mocy. Dane są przechowywane w bazie danych MongoDB, a raz dziennie generowany jest raport ze wszystkich generatorów zawierający średnią moc wygenerowaną przez każdy generator w ciągu godziny.

[link](https://github.com/kamilszwaradzki/PowerPlant)

## Hangman
<img width="1144" height="1105" alt="obraz" src="https://github.com/user-attachments/assets/2e4c8797-a286-47ef-8937-0f284dcbc131" />

### Przeznaczenie \ Purpose
**EN:** Randomly select a word from a file, have the user guess characters in the word. For each character they guess that is not in the word, have it draw another partof a man hanging in a noose. If the picture is completed before they guess allthe characters, they lose.

**PL:** Aplikacja losowo wybiera słowo z pliku i prosi użytkownika o odgadnięcie znaków w tym słowie. Za każdy wybrany znak, który nie występuje w słowie, rysuje kolejną część postaci wiszącej na stryczku. Jeśli obraz zostanie ukończony przed odgadnięciem wszystkich znaków, użytkownik przegrywa.

[link](https://github.com/kamilszwaradzki/Hangman)
